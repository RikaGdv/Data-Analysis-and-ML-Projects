# Data Science & Machine Learning Portfolio

Welcome to my project showcase!

I am building these projects to strengthen practical skills in exploratory data analysis, machine learning, and reproducible workflows.

My goal is to transition toward research-oriented and analytical roles where structured thinking, statistical reasoning, and interpretable models are essential.

---

## 📌 About These Projects

Most datasets used were sourced from publicly available platforms (e.g., Kaggle) or are synthetic for educational purposes.  

They are intended to demonstrate workflow design, preprocessing strategies, and modeling practices rather than to derive definitive real-world clinical conclusions.

### 4️⃣ Stroke Risk Prediction and Explainable AI (SHAP)

![Python](https://img.shields.io/badge/Python-3.x-blue?logo=python)
![scikit-learn](https://img.shields.io/badge/scikit--learn-ML-orange)
![XGBoost](https://img.shields.io/badge/XGBoost-GradientBoosting-green)
![Imbalanced Data](https://img.shields.io/badge/Class%20Imbalance-SMOTE-red)
![Explainability](https://img.shields.io/badge/Model-SHAP-purple)
![Status](https://img.shields.io/badge/Project-Complete-success)

**Focus areas:**

- Handling severe class imbalance (95% negative / 5% positive) using SMOTE strictly isolated to training sets  
- Hyperparameter tuning of XGBoost classifiers via GridSearchCV  
- Clinical threshold optimization prioritizing recall and biological safety over raw accuracy  
- Explainable AI using SHAP for transparent global and patient-level interpretation  

This project illustrates methodical, biologically plausible preprocessing and interpretable machine learning for high-risk clinical screening.
