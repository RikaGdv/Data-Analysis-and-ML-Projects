# Data Science & Machine Learning Portfolio

Welcome to my project showcase!

I am building these projects to strengthen my practical skills in exploratory data analysis, machine learning and reproducible workflows.

My goal is to transition toward research-oriented and analytical roles where structured thinking, statistical reasoning and model interpretation are essential.

---

## 📌 About These Projects

Most datasets used in these projects were sourced from publicly available platforms (e.g., Kaggle) and include both synthetic and educational data.  

They are used here to demonstrate methodology, workflow design, preprocessing strategies, and modeling practices rather than to derive definitive real-world clinical conclusions.

---

## 📂 Project Roadmap

Below is the chronological progression of my projects as I continue developing my skills.

---

### 1️⃣ Healthcare Data Analysis Project

My first complete end-to-end data analysis project.

Focus areas:
- Data cleaning and validation  
- Feature engineering (e.g., Length of Stay, categorical binning)  
- Exploratory visualization  
- Baseline machine learning models (Logistic Regression, Random Forest)  
- Interpretation of results in the context of synthetic data  

This project helped me understand how to structure a reproducible analytics pipeline from raw data to model evaluation.

---

### 2️⃣ Disease Symptoms and Patient Profile Analysis

In this second project, I worked with a smaller and more logically structured dataset (N = 349) to predict disease outcomes.

Focus areas:
- Symptom prevalence analysis  
- Outcome correlation exploration  
- Ordinal encoding for biologically ordered variables  
- Model comparison and feature importance interpretation  

This project strengthened my understanding of nonlinear model behavior and how feature interactions influence predictive performance.
